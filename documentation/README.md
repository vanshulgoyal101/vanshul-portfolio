# Vanshul Portfolio Documentation

This folder contains comprehensive documentation for the entire codebase, including architecture, components, hooks, styles, assets, and configuration. Use these documents for context engineering, onboarding, and future development.

## Structure
- `architecture.md`: High-level overview of the project structure and flow.
- `components.md`: Detailed documentation of all React components.
- `hooks.md`: Custom hooks and their usage.
- `styles.md`: Styling approach and global styles.
- `assets.md`: Images and public assets.
- `config.md`: Configuration files and environment setup.
- `data.md`: Static data and JSON files.

Each document is designed to be self-contained and exhaustive for its topic.
