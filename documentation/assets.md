# Assets Documentation

## Images
- Located in `public/images/projects/`
- Used for profile, project showcases, and blog posts.
- Example files: `profile.jpeg`, `nasa-herc.jpg`, `Solaride.png`, etc.

## Public Files
- `robots.txt`, `sitemap.xml`: SEO and crawler configuration
- `CNAME`: Custom domain for deployment
- SVGs: Used for icons and graphics

## Usage
Assets are referenced via relative paths in components (e.g., `<img src="/images/projects/profile.jpeg" />`).

## Extending
Add new images or assets to `public/images/projects/` and reference them in components.
