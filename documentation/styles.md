# Styling Documentation

## Global Styles
- Defined in `src/styles/GlobalStyles.js` and `src/index.css`.
- Uses CSS variables for colors, spacing, typography, and gradients.
- Provides base styles for body, headings, links, etc.

## styled-components
- Used throughout components for scoped, dynamic styling.
- Theme variables are accessed via CSS custom properties.

## Responsive Design
- Media queries are used in styled-components for mobile and tablet layouts.

## Extending
Add new global styles in `GlobalStyles.js` or extend component styles using styled-components.
